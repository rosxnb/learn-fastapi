# Introduction
The repo contains separate folders with code files and markdown note files.


## Environment Setup
```zsh
python -m venv .venv
source .venv/bin/activate
pip install fastapi "uvicorn[standard]"
```


## Run Server
```zsh
uvicorn file_name:server_name --reload
```
